# crs-astro

[MU-CRS](https://crs.mahidol.ac.th/) website, built with [Astro](https://astro.build/) and [ThaiSite](https://github.com/ThaiUI/ThaiSite).

## Setup

```
pnpm install
```

```
pnpm dev
```

## Deploy

Recommend [Cloudflare Pages](https://pages.cloudflare.com/) or [Netlify](https://www.netlify.com/) or [Vercel](https://vercel.com/).
