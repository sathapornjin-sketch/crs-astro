# crs-astro

[MU-CRS](https://crs.mahidol.ac.th/) website, built with [Astro](https://astro.build/) and [ThaiSite](https://github.com/ThaiUI/ThaiSite).

## Setup

> [!IMPORTANT]
> **Prerequisites:** Install [Node.js](https://nodejs.org/en/) (v22 or later) and [pnpm](https://pnpm.io/).

1. **Install dependencies**

```sh
pnpm install
```

2. **Start development server**

```sh
pnpm dev
```

3. Open your browser and navigate to `http://localhost:4321` to view the site.

## Deploy

Build the project for production:

```sh
pnpm build
pnpm preview
```

You can deploy the `dist/` directory to any static hosting service.
